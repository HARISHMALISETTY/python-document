📘 Module 5: Libraries, File Handling, and Exception Handling

This module introduces essential Python libraries, file operations, and error handling techniques.

1. Working with Files

  • Reading files using open() and read()
  • Writing to files using write(), writelines()
  • Modes: r, w, a, r+, b
  • Useful file methods: seek(), tell(), close()
  • Using with statement (Context Managers)

2. Exception Handling

  • try, except, else, finally
  • Handling specific exceptions like ZeroDivisionError, FileNotFoundError
  • Using the raise keyword to throw exceptions
  • Creating Custom Exception Classes

3. Useful Standard Libraries

  • math – mathematical functions like sqrt(), ceil()
  • random – random number generation, shuffling, sampling
  • datetime – date/time manipulation
  • os – directory and file path handling
  • sys – access system-specific parameters and functions

4. External Libraries

  • requests – sending HTTP requests
  • pandas – data manipulation and analysis (basic intro)
  • Installing packages with pip

5. Regular Expressions (Regex)

  • What is a regular expression?
  • Using the re module in Python
  • Common methods: re.search(), re.match(), re.findall(), re.sub()
  • Basic patterns: \d, \w, [a-z], ^, $, *, +, etc.
